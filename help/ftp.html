FTP / Web Syncing

In KeePass Touch the function FTP / Web Syncing lets you synchronize with a webserver easily. If you own e.g. a website or a server at home thats connected to the internet, you're good to go. To FTP sync you have to:

  1. Requirement: Any internet connection.
  2. Press on the globe in the bottom toolbar.
  3. Input your FTP server IP / Address in the popup textfield and acknowledge the input with the button below.
  4. Input your FTP username / password in the corresponding textfields and acknowledge the input with the button below.
  5. Select an option to either download a file from your webserver or to upload a local database file to it.
  6. Upload: Select a file from the fileslist. Then choose in the upcoming interface the folder you would like to put it in and press the top right button.
  7. Download: Navigate through the folders on the webserver and choose a file to download.

To reset the FTP Data that are saved automatically visit the settings.
