WiFi / WLAN Local Syncing

Sie können KeePass-Datenbanken lokal und kabellos ganz einfach mit KeePass Touch übertragen.
Um dies zu tun, folgen Sie diesen Schritten:

  1. Drücken Sie den 2. Knopf von links in der unten am Bildschirm sichtbaren Toolbar.
  2. Unter Ihrer Liste an KeePass-Datenbanken sollte eine Adresse / IP erscheinen, in etwa wie
    http://xxx.xxx.xxx.xxx oder
    http://***.iphone.local
  3. Öffnen Sie einen Browser Ihrer Wahl auf einem beliebigen Gerät und fügen Sie die Adresse / IP in die Adresszeile ein.
  4. Es erscheint ein Interface, welches Upload, Download, Löschen und Umbenennen aller Datenbanken Ihrer Wahl ermöglicht.
