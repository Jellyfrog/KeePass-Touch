Synchronisieren von Dateien mit Dropbox

KeePass Datenbank Dateien die in der Dropbox liegen können mit KeePass Touch synchronisiert werden, indem die integrierte Dropbox Schnittstelle genutzt wird.
Bitte beachten: Eine .key-Datei kann nicht synchronisiert werden für Ihre eigene Sicherheit. Verwenden Sie hierzu eine lokale Übertragungsmöglichkeit, z.B. Lokale WLAN Sync oder iTunes Sync.

Schritte:

  1. Öffne das Synchronisierungsmenu indem der (2. von links) Button gedrückt wird
  2. Wähle Dropbox Sync
  3. Mit Dropbox Zugangsdaten anmelden
  4. Wähle den Ordner mit dem synchronisiert werden soll
  5. Wiederhole Schritt 1 und 2
  6. Drücke "Fertig" in der oberen rechten Ecke"

Auto Sync

Nachdem Dropbox Sync ausgewählt wurde, wird gefragt, ob Auto Sync benutzt werden soll. Sollte dies bestätigt werden, wird bei jedem App Start die aktuelle Version der KeePass Datenbank mit der Dropbox Version verglichen und die aktuellste Version verwendet. Wir empfehlen dieses Feature, falls Dropbox zur Datenbanksynchronisierung genutzt wird!
