FTP / Web Synchronisation

FTP Synchronisation ermöglicht die Synchronisierung mit einem FTP / Webserver. Wenn Sie beispielsweise eine eigene Website oder einen Server besitzen, der mit dem Internet verbunden ist, kann es losgehen.
Um die FTP Synchronisation durchzuführen, folgen Sie diesen Schritten:

  1. Vorraussetzung: Bestehende Internet Verbindung.
  2. Drücken Sie auf den Globus in der Toolbar am unteren Ende des Bildschirms.
  3. Geben Sie Ihre FTP Server IP / Adresse in das erscheinende Textfeld ein und bestätigen Sie die Eingabe durch den entsprechenden Button darunter.
  4. Geben Sie den FTP Nutzernamen / Passwort in das entsprechende Textfeld ein und bestätigen Sie die Eingabe durch den entsprechenden Button darunter.
  5. Wählen Sie die Option entweder einen Download einer Datei von Ihrem Server durchzuführen oder eine Datenbank via Upload auf Ihren Server zu übertragen.
  6. Upload: Wählen Sie eine Datei aus der Dateiliste aus. Dann wählen Sie den Speicherort über das erscheinende Interface und navigieren zum gewünschten Ordner und bestätigen die Auswahl über den oben rechts angezeigten Knopf.
  7. Download: Navigieren Sie durch die Ordnerstruktur Ihres Servers und wählen Sie eine Datei zum Download aus.

Um die FTP Serverdaten zurückzusetzen, besuchen Sie die Einstellungen und wählen den Knopf "Reset FTP Data."
