Using Key Files

KeePass Touch can open KeePass 1.x/2.x files that use a key file instead of or in addition to a password.

Steps:

  • Load your KeePass database and key file in KeePass Touch using iTunes, Local Sync, etc.
  • Open your KeePass file in KeePass Touch
  • When prompted for your password you can enter a password and/or select a key file

Note: KeePass Touch will automatically select your key file if it has the same filename as your KeePass file but with a .key extension.
Additionally, please don't use any cloud sync as we do not allow the keyfile to be synced this way for your own safety
