4.1.1

Attention Due to changes in iOS 10, it is now necessary to include the NSPhotoLibraryUsageDescription in your app’s Info.plist file if you want to use HockeySDK’s Feedback feature. Since using the feature without the plist key present could lead to an App Store rejection, our default CocoaPods configuration does not include the Feedback feature anymore. If you want to continue to use it, use this in your Podfile:

ruby pod "HockeySDK", :subspecs => ['AllFeaturesLib']

Additionally, we now also provide a new flavor in our binary distribution. To use all features, including Feedback, use HockeySDK.embeddedframework from the HockeySDKAllFeatures folder.
